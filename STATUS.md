### Технические детали:

- **Backend**: Node.js + Express + Prisma + SQLite
- **Frontend**: React + Vite
- **Telegram**: telegram npm package (MTProto)
- **База данных**: SQLite (можно переключить на PostgreSQL)

### Что нужно сделать для запуска:

1. Заполнить `backend/.env` реальными данными Telegram API
2. Запустить серверы (./start.sh или вручную)
3. Авторизоваться в Telegram через /api/auth/send-code и /api/auth/verify
4. Использовать админку на http://localhost:5173
