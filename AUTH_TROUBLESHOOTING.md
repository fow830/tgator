# Решение проблем с авторизацией

## Проблема: Коды не приходят

### Возможные причины:

1. **Flood Wait** - Telegram ограничил частые запросы
   - Решение: Подождать 10-15 минут
   - Затем попробовать снова: `cd backend && node send-code.js`

2. **Код пришел, но не заметили**
   - Проверьте Telegram приложение
   - Проверьте уведомления
   - Код может быть в разделе "Telegram Login"

3. **Временная блокировка**
   - Telegram может временно блокировать частые попытки
   - Подождите 30-60 минут

### Альтернативные решения:

#### Вариант 1: Использовать существующий code hash

Если у вас есть код, который пришел ранее (даже если кажется, что истек):

```bash
cd backend
node auth-with-code.js <КОД>
```

#### Вариант 2: Подождать и попробовать снова

```bash
# Подождите 15 минут, затем:
cd backend
node send-code.js
# Проверьте Telegram и используйте новый код
```
